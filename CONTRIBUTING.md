# Contributing to Rently

Thank you for your interest in contributing to Rently! Here are a few ways you can help.

## How to Contribute

1. **Fork the Repository**: Start by forking the project to your own account.
2. **Clone the Repo**: `git clone https://github.com/your-username/Rently.git`
3. **Create a Branch**: `git checkout -b feature/your-feature-name`
4. **Install Dependencies**: `npm install`
5. **Make Changes**: Implement your feature or bug fix.
6. **Commit Changes**: Use clear and descriptive commit messages.
7. **Push to Fork**: `git push origin feature/your-feature-name`
8. **Create a Pull Request**: Submit your PR for review!

## Development Guidelines
- Ensure your code follows the existing style.
- Use `npm run lint` to check for style errors.
- Test your changes locally before submitting a PR.
